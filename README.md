## The static web application.

### Install node_modules

yarn install

### How to run on dev environement:

1. yarn install
2. yarn start

### How to run on pro environement with static server:

1. yarn install
2. yarn build
3. yarn global add serve
4. serve -s build

### How to run unit test

1. yarn install
2. yarn test

### How to run eslint

1. yarn install
2. yarn lint

### Auto format with prettier

1. yarn format
